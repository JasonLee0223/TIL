# magnitude, abs
절대값을 리턴해주는 2가지 방법

## 🅰️ magnitude
```Swift
let magnitudeNumber = -10.magnitude // 10
```

## 🅱️ abs
```Swift
let absNumber = abs(-15)            // 10
```
